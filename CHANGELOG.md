## [Unreleased]

### Added
- Release and distribution pipeline (GitHub Actions + GoReleaser)
- Multi-platform binaries: Linux/macOS (amd64 & arm64) and Windows (amd64)
- SHA-256 checksums published alongside every release
- Build-time version embedding (`version`, `commit`, `date`) in the binary
- `/version` HTTP endpoint returning build info
- End-to-end tests for the MCP orchestration flow
- `fail_step` MCP tool with retry and escalation logic
- `complete_step` MCP tool with variable extraction
- `get_next_step` MCP tool with step advancement
- `get_run_status` MCP tool for full run inspection
- `start_run` MCP tool to start workflow executions
- SQLite-backed state store for runs, steps, and variables
- Workflow loader with YAML + JSON Schema Draft 2020-12 validation
- Template interpolation (`{{variable}}`) and regex-based variable extraction
- Domain models: `Run`, `Step`, `Variable` with typed status enumerations
